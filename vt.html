Exercício de Variáveis em JavaScript

Passo 1 – Declarando variáveis

var nomeCantina = "Cantina da Escola"; console.log("Bem-vindo à " + nomeCantina);
Resultado aparecerá aqui e no console.

Passo 2 – Atualizando valores

salgados = salgados - 5; let totalVendido = 5 * precoSalgado; console.log("Agora restam " + salgados + " salgados."); console.log("A cantina vendeu R$" + totalVendido);
Resultado aparecerá aqui e no console.

Passo 3 – Teste rápido

O que acontece se tentar mudar o valor de precoSalgado?

precoSalgado = 6; // Teste e veja o erro

O que acontece se criar outra variável com var dentro de um bloco {}?

if (true) { var testeVar = "Sou var"; let testeLet = "Sou let"; console.log(testeVar); // funciona console.log(testeLet); // funciona } console.log(testeVar); // funciona console.log(testeLet); // ERRO

Conclusão

Este exercício demonstra as diferenças entre var, let e const:

  • var tem escopo global/função e pode ser redeclarada
  • let tem escopo de bloco e pode ter seu valor alterado, mas não redeclarada
  • const tem escopo de bloco e não pode ter seu valor alterado após a declaração
